Module: index.js

This file contains the implementation of a RESTful API for a movie database. It uses Express.js for routing, Mongoose for database connection, and Passport.js for authentication. The API provides endpoints for retrieving movies, genres, directors, and users, as well as adding and removing movies from a user's favorites list. It also supports user registration and updating user information.
