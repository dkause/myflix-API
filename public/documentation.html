My Flix - get your daily dose of distraction

This is an exercise for programming an API, which gives you general and detailed information about movies, directors and actors and genres. You can also add users, use a favorites list and update users information.

List of Url Endpoints

  • Return a list of ALL movies to the user;
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username)
  • Allow users to add a movie to their list of favorites (showing only a text that a movie has been added — more on this later)
  • Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed — more on this later)
  • Allow existing users to deregister (showing only a text that a user email has been removed — more on this later)
Business Logic URL HTTP Method query parameters Request body data format Response body data format
Get a list of all Movies /movies GET n/a n/a JSON object
Get data of a specific movie /movies/=title GET =title n/a JSON Object
Get data of a specific movie genre movies/genre/=genreName GET =genreName n/a JSON object
Get Data of a specific director movies/director/=directorName GET =directorName n/a JSON object
Get a list of all Users /users GET n/a n/a JSON object
Register/Create a user users/ POST n/a JSON Object string= ConfirmationMessage
Update user info /users/=username/update PUT =username JSON Object string= ConfirmationMessage
Delete User /users/=userId DELETE n/a n/a string= ConfirmationMessage
Add movie to user favorite list /users/=userId/favorites/=movieTitle PATCH =movieTitle, =userId n/a string= ConfirmationMessage
Delete movie from user favorite list /users/=userId/favorites/=movieTitle DELETE =userId, =movieTitle n/a string= ConfirmationMessage
